Tips for browsing online media safely

1
Use safety mode: This filter limits some inappropriate content, but it's not 100% effective.
2
Turn off auto-play: This prevents videos from automatically playing after one ends.
3
Report harmful content: You can report content that you think is harmful.
4
Set videos to private: If you post videos, set them to private so only your followers can see them.
5
Use Google SafeSearch: This can help keep your browsing safe.
